Mine swipper game .asm instructions:
1.the game board is built using a 10x10 array
2.At the beginning, the array will be empty.
3.Writing directly to viddeo memory B800, 640x900 resolution
4.The player can move cursor using the arrow keys up,down,right and left.
5.The player can press space key to see the mines.
6.The player can press F key to put a flag in the chosen cell.
7.The number of mines is calculated by generating a random number using the system time.
